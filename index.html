Setup

Just load the file from the jsdelivr CDN like this:
<script src="https://cdn.jsdelivr.net/gh/starke-marketing/r360-frontend-calls/calls.min.js"></script>

Auto Login

Try clicking this button, for example

Auto Login Button

To get a button working like that, you'll need something like this:

<a href="#" id="r360-login-button" data-memberid="1" data-email="dummy@email.com">Auto Login Button</a>

Where data-memberid is the Member ID from your Webflow platform and data-email is the current user email address.

  • If there is no user account associated with that Member ID, the platform will create one for it and then perform the auto login.

  • If a user account associated with that Member ID exists but the user's email address is different from the one that the platform has, the platform will update said information and then perform the auto login.

  • If a user account associated with that Member ID exists and the email address is the same, the platform will not make any changes and perform the auto login
