Auto Login

Try clicking this button, for example

Auto Login Button

To get a button working like that, just add the class login-button and the attribute data-identifier="{dynamic-identifier}" to any clickable HTML element. Where {dynamic-identifier} is a valid user identifier (you choose what to use here).

IF the identifier you choose to use is a valid email address, AND the user does not have an associated account on the platform, it will automatically create an account and link it to the email address provided and then perform the auto login.

Now to programatically signup a new user onto the r360.successionadvisors.com platform, just add the class signup-form and the attribute data-identifier="{dynamic-identifier}" to the form that performs the registration on your platform.
